## Getting started

### Dependency

- Yarn

> You can install from [Yarn Installation](https://yarnpkg.com/lang/en/docs/install/#debian-stable)


### Run this project
```bash=
git clone git@github.com:Holychung/IoT_Project.git
cd IoT_Project
yarn install
yarn run start
```
